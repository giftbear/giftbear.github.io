Conda安装与使用
2023-03-22|Linux

Conda安装

  1. Conda下载地址: https://mirrors.tuna.tsinghua.edu.cn/anaconda/miniconda/
  2. Conda安装: bash Miniconda3-latest-Linux-x86_64.sh

Conda添加仓库

conda config --add channels defaults
conda config --add channels bioconda
conda config --add channels conda-forge

Conda安装软件

conda install tmux
conda install -c conda-forge parallel
conda install -c bioconda gatk4
conda uninstall gatk4 (卸载软件)

创建与激活环境

  1. 创建环境: conda create -n project python=3.12
  2. 激活环境: conda activate project
  3. 离开环境: conda deactivate
  4. 查看环境: conda env list
  5. 删除环境: conda env remove -n project

Errors

  1. Error while loading conda entry point: conda-libmamba-solver (libarchive.so.19: cannot open shared object file: No such file or directory):
    conda install --solver=classic conda-forge::conda-libmamba-solver conda-forge::libmamba conda-forge::libmambapy conda-forge::libarchive
