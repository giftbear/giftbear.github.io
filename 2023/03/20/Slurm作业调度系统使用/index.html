Slurm作业调度系统使用
2023-03-20|Linux

基础命令

集群信息sinfo

命令 功能
sinfo -N 查看节点信息
sinfo --partition= 查看队列信息
sinfo --help 查看所有选项

作业提交sbatch

  1. 作业脚本示例:
  2. 提交作业: sbatch test.slurm
  3. 交互作业: srun -N 1 -p partition1 command
  4. 其他参数:
    参数 功能
    --time=00:00:00 运行时间
    --mem=<MB> 内存
    --mem-per-cpu=<MB> 每个cpu内存
    -n 任务数
    --cpu-per-task 每个任务cpu
    --gres=gpu:2 每个节点gpu数
  5. 批量并行提交作业:
    time cat index.cmd | parallel --tmpdir tmp -j 5 srun -p partition1 -t 24:00:00 --mem 40G -c 4 -n 1 -N 1 bash -c {} &> index.log (并行执行5个index.cmd中的命令)

作业信息squeue

命令 功能
squeue -j jobID 查看作业信息
squeue --users= 查看指定用户的作业信息

squeue | grep partition | less (查看某个队列的运行任务)

作业删除scancel

scancel jobID

作业状态scontrol

命令 功能
scontrol show job jobID 查看作业信息
scontrol hold jobID 暂停作业
scontrol release jobID 恢复作业
scontrol update dependency=jobID 作业依赖，jobID完成后开始作业

作业记录sacct

命令 功能
sacct --state= 查看指定状态的作业信息
sacct -l 查看详细作业信息

*参考资料: https://slurm.schedmd.com/documentation.html

节点，CPU，核心，线程

  • 节点: 相当于服务器
  • 一个节点可以有多个CPU，一个CPU可以有多个核心，一个核心可以有多个线程
  • 查看CPU信息: lscpu
    • 物理CPU数: cat /proc/cpuinfo |grep "physical id" |sort |uniq |wc -l
    • 每个物理CPU核数: cat /proc/cpuinfo |grep "cpu cores" |uniq
    • 逻辑CPU数(线程数): cat /proc/cpuinfo |grep "processor" |wc -l
  • 查看内存信息: free -g (-m 以兆为单位)
