MacBook Air安装Windows7系统
2021-11-28|System

制作U盘启动盘

  • 选择一款pe工具制作
  • 恢复U盘启动盘为正常U盘: cmd -> diskpart -> list disk -> select disk 1 -> clean -> 格式化

下载系统镜像

  • 下载地址: https://msdn.itellyou.cn/ (本例下载cn_windows_7_ultimate_x64_dvd_x15-66043.iso)
  • 将系统镜像放到U盘启动盘

进入PE系统

  1. 插入U盘启动盘
  2. 按住option键 (不同电脑进入的键不同)
  3. 按开机按钮
  4. 选择进入PE系统

备份数据

  • 在PE系统中将重要数据备份到U盘

安装系统

  • 格式化与分区: 根据需求进行分区, 一般都需要将安装系统的盘进行格式化 (Windows7格式化为NTFS格式)
  • 安装方式: 可以选择一键装机，也可以自己通过系统镜像里的setup安装

安装出现的问题

系统安装完后鼠标键盘不可用

  • 下载带有驱动的系统镜像
  • 下载驱动并放入U盘启动盘, 安装时添加驱动
  • 通过dism工具将驱动自己添加到系统镜像中

系统图标变大

  • 在控制面版将分辨率调高

扬声器不可用

  • 安装电脑对应版本的BootCamp: https://support.apple.com/zh-cn/HT205016 (根据电脑的型号选择对应版本下载并安装)
