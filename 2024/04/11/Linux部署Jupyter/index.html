Linux部署Jupyter
2024-04-11|Linux

安装Jupyter和IPython

pip install ipython
pip install jupyter

IPthon生成密码

ipython

配置Jupyter

jupyter notebook --generate-config
vim jupyter_notebook_config.py

运行Jupyter

nohup jupyter notebook 1>jupyter.log 2>&1 &

本地映射端口

ssh -N -f -L 8887:localhost:8887 user@serverIP
