Markdown基础语法

元素 语法
标题 # h1
## h2
### h3
有序列表 1. item1
2. item2
无序列表 - item1 或 * item2
引用 > content
代码 `code`
代码块 ```
code
```
链接 [title](https://www.example.com)
图片 ![title](image.jpg)
粗体 **text**
斜体 *text*
字体颜色 <font color=#FF0000>text</font>
分隔线 --- 或 ***

*内容不转义: 用代码块符号```或转义符\
