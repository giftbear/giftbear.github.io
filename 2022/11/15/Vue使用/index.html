安装及使用Vue

  1. 安装vue-cli脚手架: npm install -g @vue/cli
  2. 创建vue项目: vue create project_name
  3. 启动项目: npm run serve
  4. 打包项目: npm run build
  5. 安装包: npm install element-ui --save (安装的包记录在package.json)
  6. vue配置: vue.config.js
